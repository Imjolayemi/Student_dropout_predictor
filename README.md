# 🎓 Student Dropout Predictor AI

## 🌟 3MTT NextGen Showcase (Education Pillar)
This project is an early-warning dashboard designed to help teachers identify students at risk of dropping out, allowing for timely intervention.

## 🛠️ How It Works
The app takes in a student's attendance and academic scores. It then simulates an AI classification model to categorize the student's dropout risk as Low, Medium, or High, providing instant recommendations to the teacher.

## 🚀 Built With
* Python
* Streamlit (Web Framework)
* Pandas & NumPy (Data processing)

## 💻 How to Run Locally
1. Clone this repository or download the files.
2. Open your terminal and install the requirements:
   `pip install -r requirements.txt`
3. Run the app:
   `streamlit run app.py`
